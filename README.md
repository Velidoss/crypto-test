### Full stack project

to run backend:
```
cd server
npm run dev
```
to run frontend:
```
cd client
npm run dev
```

Also, provide the `.env` files:

for backend: 
```
ATLAS_USER_NAME=
ATLAS_USER_PASSWORD=
PORT=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
CLOUDINARY_URL=
```

for client: 
```
VITE_API_ENDPOINT=
VITE_CRYPTO_API_KEY=
```
